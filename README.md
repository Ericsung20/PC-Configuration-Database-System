# Intelligent PC Configuration & Compatibility Database System

A relational database project for a PC build platform that supports component management, build estimation, and hardware compatibility analysis.

## Overview
This project was developed as part of a database course project.  
The system was designed for a PC configuration platform where users can browse components, create build estimates, share posts, and check compatibility across hardware parts.

## Main Features
- User, post, estimate, and sharing data management
- Component catalog management for PC parts
- Compatibility analysis across CPU, motherboard, memory, and power constraints
- Relational schema design with normalized tables and foreign-key relationships
- SQL queries for retrieving build details and compatibility results

## Database Design
The project includes:
- Conceptual and logical database design
- Entity-Relationship modeling
- Table creation scripts
- Insert scripts for sample data
- Query examples for platform workflows

## Tech Stack
